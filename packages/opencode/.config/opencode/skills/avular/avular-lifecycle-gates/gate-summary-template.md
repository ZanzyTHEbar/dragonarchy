# Gate Summary Template

Use this when summarizing engineering readiness for a stage or gate discussion.

```markdown
# <project or change> - <stage/gate> summary

## Current stage

- Current lifecycle stage
- Next gate

## Engineering evidence

- Completed
- In progress
- Missing

## Key technical risks

- Risk
- Impact
- Mitigation

## Waivers

- Missing item
- Reason for proposed waiver
- Residual risk
- Follow-up owner

## Recommendation

- Proceed
- Proceed with waiver
- Postpone
- Re-scope
```

## Usage Notes

- Keep it short enough for decision-making.
- Link out to detailed deliverables instead of copying them into the summary.
